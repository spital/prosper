# Prosper

<br/>

## Description
Command runner with translation, calc, currency and unit conversion.

Also includes:
* Global shortcuts
* Clipboard history
* Base64 encode/decode

Supports: Mac, Windows and Linux

<br/>

Quick commands:
* `:s` to see shortcuts and change them
* `:q` Quit the app
